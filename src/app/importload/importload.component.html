Import Product Items Products / Load Items


  1. Download the Load Item Template to your local machine.
  2. Enter the Item Name in column A, row 2 under the Item Name header.
  3. If the Item uses serial numbers, enter Y in column B, row 2 under the Use Serial Numbers header.
  4. If the Item has a UPC Code associated with it, enter Y in column C, row 2 under the Use UPC Codes header.
  5. If the Item has a UPC Code associated with it, enter the Item Reference Number assigned to this item in column D, row 2 under the Item Reference Number header. If you do not use UPC Codes for this item, leave this cell blank.
  6. If you want to enable the Counterfeit Detection Module for this Item, enter Y in column E, row 2 under the Counterfeit Detection header.
  7. If the item does have Counterfeit Detection enabled, enter a corresponding radius in miles in column F under the Counterfeit Radius header. The counterfeit radius value is used to determine how many miles an individual product can move between scans without triggering a possible counterfeit notification.
  8. If you want to enable the Content Management Module for this Item, enter Y in column G, row 2 under the Content Management header.
  9. If you want to enable the Recall Management Module for this Item, enter Y in column H, row 2 under the Recall Management header.
  10. If you want to enable the Forms Management Module for this Item, enter Y in column I, row 2 under the Forms Management header.
  11. Repeat these steps for as many items as you want to add, moving down a row in the file for each new item.
  12. For more information, visit the Documentation page.
  13. Save the file in CSV (Comma delimited) format.
  14. Click the Select file button below and locate your CSV file.
  15. Click the Load New Items button to import your information.
